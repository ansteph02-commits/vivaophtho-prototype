# VivaOphtho Prototype

Bright landing page and interactive free-case prototype for VivaOphtho, an examiner-style ophthalmology viva clinical cases app.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Build

```bash
npm run build
```

## Current scope

- Bright landing page
- Exam-aware countdown and waitlist form prototype
- Interactive 8-step free case player
- Debrief and self-scoring flow
- Case library, mastery, blueprint, pricing and follow-along sections

The email form stores prototype submissions locally in the browser. It still needs a real backend or form service before public launch.
